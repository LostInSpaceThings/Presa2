# What Is Presa 2 
[Join Discord](https://discord.gg/PhsMp452zP)

Presa 2 is the successor to Presa 1. In this new modpack, we put the effort it to give you over 100 hours of new content. This includes a new dimension, new structures, and constant exploring.
